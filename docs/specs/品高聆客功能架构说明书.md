## 1. 产品定位

品高聆客（LinkPC）是品高软件推出的企业级统一工作台客户端，面向政企用户，提供：

- **统一入口**：整合即时通讯、应用集成、AI 助手、工作流审批于一体
- **安全合规**：面向政企的零信任安全接入（ZTA），支持内网区隔离
- **平台化**：通过"应用工场"开放平台，支持企业自定义应用接入
- **AI 增强**：内嵌 LinkClaw 智能体平台，提供 Agentic 工作流能力

## 2. 整体架构

### 2.1 窗口结构

品高聆客采用 Electron 多窗口架构：

| 窗口 | 类型 | 内容 |
|------|------|------|
| home.html | 主窗口（Page） | 消息列表、应用中心、顶部导航栏 |
| 项目协管 Webview | Webview | 待办/待阅/已办工作流审批（bingosoft.net） |
| AI 助手 Webview | Webview | LinkClaw 智能体平台（localhost:12258） |

### 2.2 顶部导航栏

```
[AI助手入口] [消息●] [应用 ▾]          [搜索🔍] [ZTA状态] [用户头像]
```

- **消息**：主消息列表，右上角显示未读数角标
- **应用**：应用启动台
- **搜索**：全局搜索（联系人、消息、群组、部门、服务号、应用、智能体）
- **ZTA 状态**：零信任安全状态指示器（零信任通道开关）
- **用户头像**：个人资料、设置入口

### 2.3 左侧消息过滤栏

消息列表左侧提供分类过滤：

| 分类 | 说明 |
|------|------|
| 全部 | 所有会话（含未读计数 83+） |
| @我 | 被@提及的消息 |
| 未读 | 仅显示未读会话 |
| 智能体 | AI 智能体发起的会话 |
| 单聊 | 一对一私信 |
| 群组 | 多人群聊 |
| 工作 | 工作相关会话（来自流程系统） |
| 服务 | 服务号消息（系统通知类） |
| 会议 | 会议相关消息 |

## 3. 消息模块

### 3.1 会话列表

- 显示联系人头像、姓名、最新一条消息预览、时间戳、未读计数
- 支持按日期分组展示历史消息
- 支持消息类型标记：`[图片]`、`[文件]`、`[语音]`、`[聊天记录]`、`[未读]`

### 3.2 聊天界面

聊天界面支持以下功能：

- **文本消息**：实时 IM 通信
- **@提及**：@某人触发特别提醒
- **撤回消息**：已发消息可撤回
- **文件传输**：发送文件（支持 PDF、PPT、Word、drawio 等格式）
- **图片发送**：内嵌图片预览
- **语音消息**：语音录制发送
- **聊天记录转发**：转发历史聊天记录

![聊天窗口](linkpc_screenshots/03_msg_全部.png)

### 3.3 群组功能

- 支持创建、解散群组
- 管理员可将成员加入/移出群组
- 群转让：转让群主权限
- 支持修改群名称
- 群消息：@全体成员广播

### 3.4 服务号消息

来自各业务系统的通知推送：

- 假勤系统：考勤异常审批提醒、请假申请通知
- 合同系统：合同到账通知、合同会审流程
- 工作量填报系统：填报提醒
- 绩效系统：绩效评估通知
- 报销系统：报销审批进度
- 项目协作：协管考核表通知
- 共享协作：文件共享通知
- 云票夹：发票收录通知

## 7. AI 助手（LinkClaw）

LinkClaw 是品高内嵌的企业级 AI 智能体平台，运行于本地 Web 服务（端口 12258），通过顶部导航入口进入。

### 7.1 主界面

- **问候语**："Hello，早上好" 个性化欢迎
- **安全提示**："当前为互联网区，严禁输入涉密信息，谨防泄露"
- **第三方模型警示**："第三方公有云大模型禁止上传公司敏感信息"

### 7.2 导航功能模块

#### 新建会话
发起新的 AI 对话。支持选择工作目录和 AI 模型。

**支持模型**：
- MiniMax-M2.7（当前默认）

**工作目录**：可选择本地目录，使 AI 具备文件访问能力。

**预置快捷指令卡**：

| 卡片名称 | 描述 |
|----------|------|
| **LinkClaw 智能问答** | 通用知识问答，快速上手助手 |
| **专业助手** | 虚拟设计官与效率管家 |
| **全能文档专家** | 专业级文档解析与排版能力 |

#### 配置
个性化设置 LinkClaw 行为：

| 设置项 | 选项 |
|--------|------|
| **界面显示语言** | 简体中文（可切换） |
| **AI 回复语言** | 简体中文（可切换） |
| **自定义规则（Rules）** | 用户自定义 Prompt 规则，引导 AI 回复风格 |
| **模型** | 切换底层 LLM |
| **技能（Skills）** | 管理 AI 技能插件 |
| **连接器（Connectors）** | 连接外部数据源/服务 |
| **能力包** | 扩展包管理 |
| **用量统计** | API 调用量统计 |

**规则示例**：
- "回答时请先用一句话总结核心观点，再展开详细说明"
- "如果我的问题模糊，请先提出 1-2 个澄清问题再作答"
- "请保持简洁，避免不必要的客套话"

还支持读取 **AGENTS.md** 和 **工作目录** 自定义上下文，用于项目级规则注入。

#### 会话
查看和管理历史对话：
- 按时间分组（今天 / 4月8日 / 7天内 / 更早）
- 每条会话显示首条提问作为标题
- 可继续已有会话

**已记录的历史会话示例**（显示了真实使用场景）：
- "请打开浏览器，访问 bing.com"（浏览器控制能力）
- "请创建一个会议"（会议系统集成）
- "根据客户在「金融机构安全管理」这个领域的业务情况，进行「全领域本体建模」"（企业专业任务）
- "请收集整理「面向对象」的基本理论，输出为 Markdown 文档"（文件写作能力）

#### 专家
管理和使用 AI 专家角色（类 GPTs）：

**系统预置专家**：
| 专家名称 | 描述 |
|----------|------|
| 代码审查专家 | 专业代码审查，发现潜在问题，确保代码规范 |
| 技术文档撰写专家 | 产出高质量 API 文档、设计文档、用户手册 |
| 前端开发工程师 | React/Vue/Angular 框架开发、UI 组件、性能优化 |
| 产品规划专家 | 产品规划、用户研究、竞品分析、产品策略 |
| 招聘助手 | 招聘场景专属助手 |

**操作**：
- 手动创建自定义专家
- 从市场添加专家

#### 创作
Prompt 驱动的创作与应用开发平台，覆盖全场景：

**快速创作模板**：
| 类型 | 名称 | 描述 |
|------|------|------|
| 灵感 | HTML 原型快速开发 | 使用 AI 快速生成 Web 原型 |
| 应用 | 数据仪表盘 | 创建可视化数据展示页面 |
| 应用 | 表单页面 | 快速生成各类表单页面 |
| 技能 | 代码审查助手 | 自动审查代码，发现问题 |
| 技能 | 文档生成器 | 自动生成技术文档和注释 |
| 技能 | 测试用例生成 | 根据代码自动生成单元测试 |
| 技能 | 多语言翻译助手 | 快速翻译技术内容 |

**创作产物类型**：
- **Web 应用**（App）：独立运行的网页应用
- **技能**（Skill）：可复用的 AI 技能插件

## 10. 动态

动态是企业内部社交 Feed 流，展示成员的工作活动，通过"应用▾"下拉菜单进入。

![动态](linkpc_screenshots/module_动态_01.png)

### 10.1 动态过滤标签

左侧导航：

| 标签 | 说明 |
|------|------|
| 全部 | 所有关联动态 |
| 关注 | 已关注成员的动态 |
| 精华 | 管理员标记的精华内容 |
| 分享 | 分享类动态（文章/链接等） |
| 工作 | 与工作项相关的操作动态 |
| 收藏 | 我收藏的动态 |
| 报障 | 报障/工单相关动态 |
| @我 | 提及我的动态 |
| 评论 | 他人对我内容的评论 |
| 赞我 | 获得点赞的动态 |

### 10.2 动态内容类型

每条动态包含：发起人、所属部门、关联项目/对象、操作描述、时间戳，以及互动按钮（**点赞 / 评论 / 收藏 / 更多**）。
